# Black Hole Strategy Game 🎮

A multiplayer strategy game where the goal is to achieve the lowest score.

## 🚀 Features
- Multiplayer gameplay
- Interactive UI
- Real-time updates using Supabase

## 🛠 Tech Stack
- React + Vite
- TypeScript
- Supabase
- Tailwind CSS

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

## 🌐 Deployment
Deployed using Vercel
